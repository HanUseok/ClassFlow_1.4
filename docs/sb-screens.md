# 화면 정의서 (페이지 단위)

- 기준일: 2026-03-06
- 기준: `app/**/page.tsx` 실제 구현

## `/`
- 목적: 루트 진입 처리
- 동작: 즉시 `/teacher`로 리다이렉트

## `/teacher`
- 목적: 교사 대시보드
- 핵심 UI: 세션 엔진 카드, 마감 카드, 준비 상태 카드, 관찰 필요 학생, 최근 근거
- 주요 액션: 대기 세션 진입/종료, 새 세션 생성, 학생 상세 이동

## `/teacher/sessions`
- 목적: 세션 목록 관리
- 핵심 UI: 필터(타입/상태/검색), 세션 카드 목록
- 주요 액션: 상태별 진입, 세션 삭제, 전체 삭제, 새 토론/발표 세션 생성

## `/teacher/sessions/create`
- 목적: 세션 생성/수정
- 핵심 UI: 주제 입력, Debate 단계 UI, 발표 설정 UI
- 주요 액션: 저장, 세션 실행

## `/teacher/sessions/[id]`
- 목적: 세션 실행/운영
- 핵심 UI: 타입별 실행 화면(토론/발표)
- 주요 액션: 진행 제어, 종료, 보고서 이동

## `/teacher/sessions/[id]/report`
- 목적: 교사용 세션 리포트 확인
- 핵심 UI: 세션 메타, 학생/로그 기반 요약
- 주요 액션: 세션 목록으로 복귀

## `/teacher/students`
- 목적: 학생 탐색
- 핵심 UI: 검색, 학급 필터, 학생 카드 그리드
- 주요 액션: 학생 상세로 이동

## `/teacher/students/[id]`
- 목적: 학생별 워크스페이스
- 핵심 UI: 학생 프로필, 세션별 기록, 필터
- 주요 액션: 기록 확인, 세션 단위 확장/축소

## `/teacher/settings`
- 목적: 명단/스테이션 설정 UI
- 핵심 UI: 탭(명단/스테이션), 편집 UI
- 주요 액션: 역할 변경, 삭제, CSV 업로드 버튼(UI)

## `/station`
- 목적: 스테이션 참여 및 토론 진행
- 핵심 UI: 대기/입장/조선택/배치/live 화면
- 주요 액션: 참여자 입장, 발언 시작/종료, 토론 종료

## `/station/report`
- 목적: 토론 리포트 조회/관리 뷰
- 핵심 UI: report/manage 전환, 로그 요약, 복귀 버튼
- 주요 액션: 뷰 전환, Teacher 또는 Station 화면으로 복귀
